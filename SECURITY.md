# Security

## How scripts execute

All Login Enterprise scripts generated by these skills **execute locally on your machine**. The skills themselves do not make network calls, access remote services, or transmit data.

When using `login-enterprise-script-runner`, the Login Enterprise standalone engine launches real applications in your Windows session. Be aware that:
- Desktop scripts open and interact with real applications
- Web/Playwright scripts open real browser sessions
- The engine may close applications when scripts complete

## What this repo does NOT contain

- No credentials, API keys, or secrets
- No Login Enterprise binaries (users supply their own ScriptEditor installation)
- No telemetry or analytics

## Reporting a vulnerability

If you discover a security issue in this repository, please **do not** open a public GitHub issue.

Instead, report it privately:
- Use [GitHub's private vulnerability reporting](https://github.com/LoginVSI/loginvsi-skills/security/advisories/new)
- Or email: security@loginvsi.com

We will acknowledge your report within 3 business days and work with you on a fix.

## CI security checks

This repository's CI pipeline includes:
- Secret scanning (API key patterns, tokens, passwords)
- Internal reference scanning (prevents accidental exposure of internal infrastructure)
- File reference verification (ensures SKILL.md links resolve to real files)
